Uses

A list of all the things I use and why.

Software
  • Emacs is my main text editor. Although I am an evil-mode user since vim kebindings are second nature to me.
  • NixOS. All of my machines currently run some variation of NixOS. I've been using linux as my primary operating system since 2019 and there's no chance of that chaning anytime soon. Nix in particular is super nice because it uses a configuration file to setup an entire computer declaratively. The language used in these files, however, could definitely use some work.
  • KeepassXC. A password manager is life changing nowadays. I've been using Keepass for quite some time and it has been pretty good. The ability to generate unique passwords for any account I need and not having to remember said password is so releiving.
Hardware
  • Lenovo X1 Carbon Gen 5. My current laptop. It still holds up well and I plan to use it for many more years.
